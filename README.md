Titre du projet : Monde des blocs(Blocks World)   
Catégorie du projet : Intelligence Artificielle  

Année :2024  
Projet en : binome/ 1 Mois  


Le projet *Monde des blocs* est un cas classique en intelligence artificielle
utilisé pour illustrer les algorithmes de planification. Il consiste à modéliser
des configurations de blocs empilés, à définir des contraintes, à planifier des séquences d’actions permettant d’atteindre un état final et à extraire des connaissances à partir des états générés.
Le projet est structuré autour de quatre axes principaux :
- Variables et contraintes  
- Planification  
- Problèmes de satisfaction de contraintes  
- Extraction de connaissances  

Technologies utilisées : Java  

Compétences acquises : 
Intelligence artificielle, modélisation par contraintes,
résolution de CSP, extraction de connaissances
